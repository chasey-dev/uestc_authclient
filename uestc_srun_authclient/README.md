# go-nd-portal
Go 你电 portal

## 安装

你可以下载 [已经编译好的二进制文件](https://github.com/fumiama/go-nd-portal/releases).

从源码安装:
```bash
$ go install github.com/fumiama/go-nd-portal@latest
```

## 使用方法

> 也可不带参数运行，会在启动时询问参数

```bash
./go-nd-portal -n 20xxxxxxxxxxx -p password [-x]
```
默认值：
 * `-ip`: 本机公网出口，可自定义
 * `-x`: 是否使用电信登陆（否）
